Day2

Day2(dataLines, isLoggingEnabled)

The Day2 class represents a game with a series of sets of colored cubes.
Constructor

# new Day2(dataLines, isLoggingEnabled)

Creates an instance of the Day2 class.
Parameters:
Name Type Description
dataLines Array.<string> An array of strings, each representing a game. Each game is represented as a series of sets of cubes, separated by semicolons. Each set of cubes is represented as a series of cube counts, separated by commas. Each cube count is represented as a number followed by a color (red, green, or blue).
isLoggingEnabled boolean A boolean indicating whether logging is enabled.
Properties:
Name Type Description
dataLines Array.<string> An array of strings, each representing a game. Each game is represented as a series of sets of cubes, separated by semicolons. Each set of cubes is represented as a series of cube counts, separated by commas. Each cube count is represented as a number followed by a color (red, green, or blue).
isLoggingEnabled boolean A boolean indicating whether logging is enabled.
maxLineLength number The maximum length of a line in the dataLines array.
availableCubes Object An object representing the available cubes, with properties for red, green, and blue cubes.
totalSum Object An object representing the total sum of cubes, with properties for part1 and part2.

Example
const dataLines = [
  "Game 1: 3 blue, 4 red; 1 red, 2 green, 6 blue; 2 green"
];
const day2 = new Day2(dataLines);
console.log(day2);
