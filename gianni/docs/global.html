# replaceLastOccurrence(sourceString, substringToReplace, replace) → {string}

This function replaces the last occurrence of a substring in a source string with a new string.
Parameters:
Name Type Description
sourceString string The original string. For example, 'Hello, world!'.
substringToReplace string The substring to be replaced. For example, 'world'.
replace string The string to replace the substring with. For example, 'GitHub Copilot'.
