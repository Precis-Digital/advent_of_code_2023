Day3(dataLines, isLoggingEnabled)

Constructor

# new Day3(dataLines, isLoggingEnabled)

This class is used to process a string of data lines and perform various operations on it. It includes methods to generate a range of numbers, determine positions to check based on a number, check if a number is part of any two-part gears, calculate the sum of the product of two-part gears, and process the numbers and update the potential gears and total sum.
Parameters:
Name Type Description
dataLines string The data lines to process.
isLoggingEnabled boolean A flag indicating whether logging is enabled.
Properties:
Name Type Description
dataLines string The data lines to process.
isLoggingEnabled boolean A flag indicating whether logging is enabled.
gears Array An array to store the gears.
totalSum number The total sum of the product of two-part gears.
